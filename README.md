# De Patria a Patria

Un proyecto web responsivo que celebra las historias y culturas de diferentes lugares del mundo, mostrando las ciudades natales de algunos empleados y estudiantes de TripleTen.

## 🌍 Descripción del Proyecto

"De Patria a Patria" es una página web interactiva que presenta una galería de ciudades de todo el mundo, cada una con su propia historia única y características culturales. El proyecto está diseñado para ser completamente responsivo, adaptándose perfectamente a dispositivos desktop, tablet y móvil.

## 🎯 Características Principales

- **Diseño Responsivo**: Optimizado para desktop (1280px+), tablet (768px-1279px) y móvil (320px-767px)
- **Interfaz Moderna**: Diseño limpio y atractivo siguiendo las mejores prácticas de UX/UI
- **Galería Interactiva**: Presentación visual de diferentes ciudades del mundo
- **Navegación Intuitiva**: Estructura clara y fácil de usar
- **Optimización Web**: Código semántico y accesible

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica y accesible
- **CSS3**: Estilos modernos con Flexbox y Grid
- **Metodología BEM**: Organización y nomenclatura de clases CSS
- **Git**: Control de versiones
- **GitHub Pages**: Despliegue y hosting

## 📱 Breakpoints de Diseño

- **Desktop**: 1280px y superior
- **Tablet**: 768px - 1279px
- **Móvil**: 320px - 767px
